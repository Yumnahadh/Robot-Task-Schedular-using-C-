# Robot-Task-Schedular-using-C-
A C++ robot task executor and scheduler is performed using the concepts of data structures and Object oriented Programming (OOP)
Main Concepts used were Queue, Stack , Linkedlist , Binary search tree and templates.
**Queue(FIFO)** :The concept of queue is used for entering the tasks and executing them 
in a FIFO order. 
**Stack(LIFO)**:This concept were used for adding the urgent task by interrupting the queue and executing the urgent task first by executing them by LIFO order
**Linkedlist:** To storing the history of the tasks being performed and completed ones. 
**Binary Search tree:** Trees are being used for searching the task completed in the list or not . 
**Template:** Making the task executor generic for different task types.
